Overview

  Thank you for your interest in our product.  Our Pos is a complete POS system developed using Laravel and the Flutter Framework. Laravel (the PHP Framework) is a web application framework with an expressive, elegant syntax. We've already laid the foundation, freeing you to create without sweating the little things. The Flutter framework created by Google is open-source mobile application development. It is used to develop applications for Android and iOS, as well as being the primary method of creating applications. Our Pos comprises mobile applications and a web panel.  This application is for Android and iOS using Flutter. A web panel was developed using Laravel.
